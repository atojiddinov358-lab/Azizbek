🌙 Orzu

Tun jimjit, yulduzlar kuladi,
Oy nur sochib, yo‘limni biladi.
Qalbimda bir go‘zal orzu bor,
U meni uzoqlarga chorlaydi.

Hayot — ba’zan yomg‘ir, ba’zan bahor,
Ba’zan quvonch, ba’zan esa g‘am-u zor.
Ammo inson umid bilan yashar,
Har tong ortida yashirin bir bahor.

Yiqilsang ham, yana tur, ey dil,
Orzularing bo‘lmasin hech ham til.
Chunki eng go‘zal kunlar hali oldinda,
Ishon — sening hikoyang endi boshlanadi. 🌟
